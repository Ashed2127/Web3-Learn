# Web3-Learn

A lightweight JavaScript project for learning blockchain fundamentals, hashing, and proof verification.

## Overview

This repository contains examples and tests for:

- `intro.js` — basic blockchain learning exercises
- `sha256-hash.js` — SHA-256 hashing utilities
- `keccak256-hash.js` — Keccak-256 hashing utilities
- `proof.js` — proof generation or verification logic
- `proof.test.js` — proof validation tests
- `sha256-hash.test.js` — hash function tests

## What You Can Learn

- hashing algorithms used in blockchain systems
- proof creation and validation patterns
- JavaScript testing with a simple repo structure

## Usage

1. Install dependencies:

```bash
npm install
```

2. Run tests:

```bash
npm test
```

## Notes

This repo is ideal for beginners exploring cryptographic hashing and blockchain proof concepts.
